Public interface

This the project that I use to master the Julia programming language. I have taken the Observer Pattern as the goal to implement. The main objects are Publisher and Subscriber. The Publisher delivers a message to the Subscribers. The information consists of temperatures that were measured.

Exports

API methods

  • createSubscriber, createPublisher, createMessage, sendMessage

Domain objects

  • Subscriber, Publisher, Message

SubscriberType values

  • MEANCALCULATOR, STDCALCULATOR, PLOTTER

PublisherType values

  • NEWSPAPER, MAGAZINE, SOCIAL_MEDIA

createSubscriber( name [, email [, subscribertype]] )

julia> daisy = createSubscriber( "Daisy", "daisy@duckcity.com" )

Subscriber("Scrooge McDuck", "scrooge@duckcity.com", PLOTTER::SubscriberType = 2)
